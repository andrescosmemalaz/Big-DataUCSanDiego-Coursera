# Big-DataUCSanDiego-Coursera
Curso de Big Data

Tiene el objetivo de Comprender los conocimientos que pueden aportar los macrodatos
experiencia práctica con las herramientas y los sistemas utilizados por los
científicos e ingenieros de big data. Del mismo modo, desarrollando  código,
experimentar cómo se puede realizar el modelado predictivo y
aprovechar el análisis de gráficos para modelar problemas. Esta especialización
preparará para hacer las preguntas correctas sobre los datos, comunicarse
 eficazmente con los científicos de datos, y hacer la exploración básica de grandes
complejos conjuntos de datos. En el proyecto final de la especialización, desarrollado en
con la empresa de software de datos Splunk, aplicarás los conocimientos
aprendidas para realizar análisis básicos de big data.
